# VOC 分割标注平台

## 启动

```bash
python3 app.py 8000
```

浏览器打开：

```text
http://127.0.0.1:8000
```

局域网共享访问：

```bash
python3 app.py 8000 0.0.0.0
```

同一局域网里的其他电脑打开这台机器的内网 IP，例如：

```text
http://192.168.1.111:8000
```

## 数据目录

- 图片：`data/images`
- 标签：`data/labels`
- 类别名：`data/classes.json`

标签按当前数据集格式读写：每行一个分割对象，格式为 `类别 x y x y ...`，坐标为 0 到 1 的归一化点位。

## 功能

- 读取同名图片和 txt 标签并展示分割点
- 新增多边形、拖动点、双击边插入点
- 删除点、删除对象、修改类别
- 删除当前图片，图片和同名标签会移动到 `data/__delete__`
- 预先配置 `0`、`1`、`2` 等数字类别对应的标签名称
- 保存当前标签到 `data/labels`
- 下载当前 txt 或导出全部标签 zip

## 快捷键

- `D`：上一张
- `F`：下一张
- `N`：绘制新多边形
- `V`：回到选择模式
- `Enter`：完成绘制
- `Esc`：取消绘制
- `Delete` / `Backspace`：删除选中的点或对象
- `Shift+Delete`：删除当前图片
- `Ctrl+Z`：撤回刚才的操作
- `Ctrl+S`：保存当前标签
